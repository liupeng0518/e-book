大部分容器镜像启动后，容器内部市区都是UTC时间，会与我们当前的时区有差别，与北京时区相差8个小时，这会导致我们排查问题以及收集容器内部的日志等创建时间与实际有出入。
这里有几种解决方法：

1. 修改镜像时区配置文件，并重新打包镜像

2. 挂载宿主机时区配置文件 -/etc/localtime- 使用volume方式挂载：
```yaml
volumeMounts:
  - name: host-time
    mountPath: /etc/localtime
    readOnly: true
  volumes:
  - name: host-time
    hostPath:
      path: /etc/localtime

```


3. 添加env

template.spec.containers 添加

```yaml
- env:
  - name: TZ
    value: Asia/Shanghai
```
